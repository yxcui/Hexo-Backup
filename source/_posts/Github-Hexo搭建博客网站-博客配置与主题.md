### 添加tags、author等页面：
`$ cd hexo`
`$ hexo new page tags`

### 更新npm版本
`$ nmp i nmp -g`

### 添加RSS插件
`$ npm install hexo-generator-feed`

### 添加search页面
`$ npm install --save hexo-generator-search`
